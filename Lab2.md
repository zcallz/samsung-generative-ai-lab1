Lab 2: Vibe Coding a Personal Portfolio

This document contains the deliverables for Lab 2, where I utilized "Vibe Coding" via Generative AI to create a personal portfolio website.

1. Defining the Persona

Role: Data Analyst

Strengths: Data Visualization, Statistical Analysis, Problem Solving

Vibe/Theme: Modern, analytical, clean, and high-tech (Dark slate & Neon Blue themes), reflecting my focus on turning raw data into actionable insights.

2. Generating Assets (Image Generation)

To match the data-focused professional persona, I conceptualized the following visual assets using AI image generation concepts:

Profile Picture Prompt: > "Professional portrait of a data analyst in a modern office, confident and approachable"

Background Image Prompt: > "Abstract technology background with glowing blue data streams, charts, and interconnected nodes, 8k resolution"

3. Web Development (Vibe Coding with AI)

Using the defined persona and assets, the entire website was generated using Generative AI.

Technology Stack: HTML5, Tailwind CSS (via CDN), FontAwesome (for icons).

Process: I instructed the AI to build a single-page portfolio with a dark/neon-blue aesthetic suitable for a data professional. It successfully structured the layout, applied Tailwind utility classes for responsive design, and integrated the generated images directly into the code. The content was tailored to highlight key data analysis strengths.
